Vamos ver alguns atalhos legais do vs code que vao deixar a nossa vida mais alegre, supimpa e feliz.

ctrl + espaço chama o autocompletar de um contexto

alt+z: Quebra automática de linha

alt+setas cima/baixo: move a seleção para cima / baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor está.

shift + alt + setas cima/baixo: copi a seleção para cima / baixo no arquivo. Se nada estiver selecionado, copia a linha em que o cursor está.
